## Multimodel-AI-Chatbot

Here's a description you can use for your README, GitHub repo, or app listing:

**Multi-Model AI Chatbot**

An interactive, fast, and visually engaging chatbot built with Streamlit and LangChain that lets users chat with multiple leading AI providers — Groq, Google Gemini, and OpenAI — all from a single interface. Users can switch between providers and models on the fly, enter their own API key, and get real-time streaming responses for a smooth, ChatGPT-like experience.

#Key features include:

A clean, modern UI with custom-styled chat bubbles, distinct avatars for the user and the AI assistant, a rotating tagline, and a CSS-powered image slideshow for a polished landing experience. A session-based chat history panel keeps track of previously asked topics for quick reference. The app supports streaming responses, so answers appear word-by-word as they're generated, leveraging Groq's ultra-fast LPU inference, Gemini's multimodal reasoning, or OpenAI's GPT models depending on user preference.

Built with: Python, Streamlit, LangChain (langchain-groq, langchain-google-genai, langchain-openai).
